# Manual de Uso - Sistema de Fluxo de Caixa

## Como usar o sistema

### 1. Registrar Receitas
- Selecione a categoria da receita
- Digite o valor (sempre positivo)
- Escolha a data
- Adicione uma descrição (opcional)
- Para cancelar uma receita, marque "Cancelamento de Receita"

### 2. Registrar Despesas
- Selecione a categoria da despesa
- Digite o valor (sempre positivo - o sistema converte automaticamente)
- Escolha a data
- Adicione uma descrição (opcional)
- Para cancelar uma despesa, marque "Cancelamento de Despesa"

### 3. Visualizar Relatórios
- Use os filtros de data para análises específicas
- O gráfico mostra a evolução do fluxo de caixa
- O resumo financeiro apresenta totais do período

### 4. Exportar Dados
- Clique em "Exportar" para baixar um arquivo CSV
- O arquivo contém todos os movimentos registrados
